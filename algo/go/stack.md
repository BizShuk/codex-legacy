# Stack

## Monotonic Stack/Queue

Every time the stack check, it will store a value for increment or decrement value from current processed chain.
And check some logics with it later.

Example: 3, 1, 2, 6, 9, 7

Stack    => 3, 1, 2, 6, 9, 7
MinValue => 3, 1, 1, 1, 1, 1
MaxValue => 3, 3, 3, 6, 9, 9
